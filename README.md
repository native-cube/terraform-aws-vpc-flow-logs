# terraform-aws-vpc-flow-logs
Terraform module for enabling AWS VPC flow logs.

## Usage

```hcl
module "vpc-flow-logs" {
  source = "native-cube/vpc-flow-logs/aws"
  version = "~> 1.0.0"

  name_prefix = "native-cube-example"
  vpc_id      = "vpc-123456789"

  traffic_type = "ALL"

  tags = {
    Project = "native-cube"
  }
}
```
